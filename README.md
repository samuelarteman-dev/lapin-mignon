# Lapin Panique

Un fond d'ecran mobile interactif avec un petit lapin rose cartoon qui gambade tranquillement. Quand on le touche, il panique et fuit en courant !

## Fonctionnalites

- Animation en boucle du lapin (idle)
- Touch/click pour declencher la fuite paniquee
- Export en GIF anime
- Export en JPEG statique

## Stack

- React + Vite
- Tailwind CSS v4
- Canvas API
- gif.js

## Developpement

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
